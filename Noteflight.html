Overview

Noteflight is an online music notation software and social music platform, designed to make traditional music notation accessible to everyone.

Projects

Reporting System

My first project was to refactor and improve Noteflight's internal reporting system, and create a dashboard for viewing monthly user stats and financial information. The existing reporting system, written in Ruby, generated a CSV with the relevant information. I refactored this reporting system to create a more general data structure with reporting information, and can be used with any sort of formatter. I also created a web formatter for this information with HTML and ERB templating, which generates a user-friendly viewable report of the most relevant information.
