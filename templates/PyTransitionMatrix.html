PyTransitionMatrix

Simple Markov Transitions

PyTransitionMatrix is a small Python class which handles data collection and transition generation for a Stochastic Matrix

PyTransitionMatrix on Github!

An example of PyTransitionMatrix in use can be found here, where it is used to manage the data collection and output generation. PyTransitionMatrix started as part of this project; I separated it into an independent library when I realized that an object which handles this might be useful in other contexts.
