ClassTeX

Simple. Clean. Versatile.

ClassTeX is a LaTeX template formatted for quick, clean, and clear note-taking.

  • Designed for use with XeLaTeX, likely compatible with other compilers (untested, but not designed to require extensive font support)
  • Includes predefined macros for creating text boxes for emphasis
  • Predefined shortcuts for common terms used when taking notes in CS/Math/Engineering classes

Download and Use

To use ClassTeX, simply download the .cls file available here and create a new TeX document of documentclass ClassTeX. For more information, check out the user guide available here.
