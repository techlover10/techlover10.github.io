Tagger

Python Library for Biased Text Generation

Tagger is a library that analyzes a body of text, and saves stylistic data in a JSON file that resembles a Stochastic Matrix that has not been normalized. This tool is useful for collecting information on a body of training data, which can then be used to generate text that roughly imitates the style of the training data.
