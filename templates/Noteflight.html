Noteflight

Software Development Intern at Noteflight | Somerville, Massachusetts

Summer 2016

Overview

Noteflight is an online music notation software and social music platform, designed to make traditional music notation accessible to everyone. Noteflight is now owned by Hal Leonard Corporation, the world's largest sheet music distributor.

Projects

Reporting System

My first project was to refactor and improve Noteflight's internal reporting system, and create a dashboard for viewing monthly user stats and financial information. The existing reporting system, written in Ruby, generated a CSV with the relevant information. I refactored this reporting system to create a more general data structure with reporting information, and can be used with any sort of formatter. I also created a web formatter for this information with HTML and ERB templating, which generates a user-friendly viewable report of the most relevant information.

Search Indexing Project

General

My second project (which I, along with another intern, spent most of the internship working on) was developing and implementing a more robust score search experience to replace the current implementation on Noteflight. The goal was to create a richer search experience for users, and allow them to discover new music using a faceted search system through which they could discover pieces by instruments, genres, and other attributes. The system was implemented in concept in various test environments, and eventually refactored and integrated into Noteflight's new content libraries system, due to be released this fall.
