(un)Intelligent README Generation for Intelligent Developers

The average developer spends 10-20 minutes documenting each of his/her software projects. On average, this amounts to more than 300 hours spent writing documentation, hours which could have spent developing innovative new solutions or expanding existing projects. ForkMe takes the stress out of writing clear, coherent READMEs and documentation, by doing the work for you!

Features

  • One-click copy as Markdown for pasting into a document
  • Intelligent auto-generation based on a large library of well-written READMEs obtained from various open-source projects
  • Built-with list is randomly selected from only the best developer libraries
  • Code snippets pulled from Enterprise FizzBuzz, one of the most well-maintained open-source projects
  • Powered by Tagger
